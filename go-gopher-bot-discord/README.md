## What is this about?
Learning how to make a discord bot in Go

## Key steps
- go mod init github.com/algebra2boy/GoGo/go-gopher-bot-discord
    - to manage dependency
- go get github.com/bwmarrin/discordgo
    - this downloads the DiscordGo package
